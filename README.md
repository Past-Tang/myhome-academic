## 概述

基于 Flask 的学术个人主页生成系统，使用 SQLite 存储个人信息、论文、项目等学术数据，自动生成专业的学术个人主页。

## 功能特性

- **个人信息管理** -- 姓名、简介、研究方向等
- **数据持久化** -- SQLite 数据库存储
- **Web 展示** -- Flask 渲染学术主页

## 使用方法

```bash
pip install -r requirements.txt
python run.py
```
